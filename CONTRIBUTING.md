# Contributing

Thanks for your interest in contributing to `create-express-init` — welcome!

Guidelines
- Use the `main` branch for releases and default development.
- Open a feature branch named `feat/<short-desc>` or fix branch `fix/<short-desc>`.
- Follow Conventional Commits for commit messages: `feat:`, `fix:`, `chore:`, `docs:`, `refactor:`.

Pull Request process
- Fork the repo and create a branch for your change.
- Push your branch and open a PR describing the change and linking related issues.
- Ensure CI passes: `npm ci && npm test`.

Local development
- Install dependencies: `npm ci`
- Run tests: `npm test`
- Lint: `npm run lint`

Thank you — your contributions make this project better.
